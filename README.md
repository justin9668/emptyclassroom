# EmptyClassroom

https://emptyclassroom.com

Find empty classrooms to study in at Boston University.

## Features

- View all available time slots for each classroom
- Search by building code, room number, or both
- Refresh to get the latest availability
- Covers CAS and CGS buildings with 70+ classrooms
